Important Concepts

The Workings of the Web

HTML stands for Hypertext Markup Language
. Basically when we go on the web, we send an HTTP request through the internet to servers, which host files.

HTML

HTML is made up of text content, markup, references to other docs and links to other pages.

HTML Markup

This is basically a bunch of tags that you use while you code.

General format:

< NAME>CONTENT</NAME>

The first tag is called the opening tag and the last tag is called the closing tag. The entire thing is called an element.

HTML Attributes

<TAG ATTR="value")contents</TAG>

Anchor - <a>
anchors are for making links
<a href="http://www.reddit.com">example link</a>
